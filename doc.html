Firefly v1.0

Довідка

FireFly - візуальний редактор CSS-анімації, що значно спрощує створення нової анімації або ж редагування вже існуючої.

Нижче описано детальну інструкції як користуватися даним редактором.

Вкладка "Клас"

  1. Елемент для пошуку необхідної анімації
  2. Кліком миші анімації буде перенесена до редактора
  3. Значок для задання значення "безкінечного повторення"
  4. Випадаючи списки статичних параметрів анімації
  5. Таби для переходу між вкладками

Крива Безьє

  1. Елементи керування
  2. Показники прогресу відносно часу
  3. Кнопки зберження чи відміни дій

Вкладка "Кадри"

  1. Додавання нових керуючих елементів для введення
  2. Після наведення на елемент, з'являється значок для його видалення

Текстовий редактор

  1. Кнопки копіювання, імпорту та експорту текстового файлу
  2. В редакторі можна вносити зміни в анімацію, вони застосуються у всіх частинах редактора

Часова лінія

  1. Кнопки керування відтвроенням анімації
  2. Індекатор прогресу
  3. Неактивний кадр
  4. При наведені на активний кадр з'являється значок для його видалення
  5. Вигляд активного кадру

Зона попердньго перегляду

  1. Елемент що анімується
  2. Редагування стилів та структури анімованого елемента
  3. Кнопка застосування змін
  4. Повернення до початкового вигляду елемента

Меню сайту

  1. Авторизація через популярні сервіси
  2. Зміна розташування елементів в редакторі
  3. Виклик вікна з інформацією про автора сайту
  4. Детальна документація
  5. Виклик вікна з підказками в користуванні сайтом

Меню сайту після авторизації

  1. Кнопка збреження анімації в базу даних для авторизованого користувача
  2. Відображення збережених раніше анімацій
  3. Дані про активного користувача
  4. Значки для прив'язання інших сервісів авторизації
  5. Вихід з профілю
